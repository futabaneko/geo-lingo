ベンガル文字の超速カンペ

ここでは、地名を読むための最低限のヒントだけを載せています。

  • 「া」= a の長音（例: ক + া = কা = kā）
  • 「ো」= o（例: ক + ো = কো = ko）
  • 「ি / ী」= i / ī（例: ক + ি = কি = ki、ক + ୀ = কী = kī）
  • 「ু / ূ」= u / ū（例: ক + ু = কু = ku、ক + ୂ = কূ = kū）
  • 固有名詞では母音が省略されることあり（例: সিলেট = Sylhet など）
